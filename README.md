   ## 程序出错怎么办
   ### 进入本仓库提交issue，格式如下
  ```Markdown
# 程序出错
## 我的电脑系统是：windows xx
## 我的cpu是：xxx
## 我的计算机架构是：XXX
## 我在程序崩溃前做了什么操作
## 我的程序是xx版本的
## 程序为Preview/Release版
```
## 如何提交涂装
```Markdown
# 提交涂装
##注册号：
##航空公司：
##现实是否存在：
##原链接：
```
接下来提交附件，如果是你制作的，可以省略原链接，否则要提供
